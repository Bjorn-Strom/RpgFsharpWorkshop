# RpgFsharpWorkshop

## I just want to see it!

Well, you can go here: [http://troykershaw.github.io/RpgFsharpWorkshop/]()

## Building
Run

```
build.sh
```

or

```
build.cmd
```

then

```
build.fsx
```

## Viewing

- Open your browser to [http://localhost:8100/index.html]()
- Press `s` to view slide notes
- Press `f` to go fullscreen
